# 请求数据

## axios-基本使用

### 安装

```bash
npm install axios
```

### 执行请求

```js
axios({
	method: 'GET', // 请求方法
	url: 'http://localhost:3000/users', // 请求路径
    params: {}, // Query 参数
    data: {}   // Body 参数
}).then(function (res) {
    // res 是响应对象
}).catch(err => { 
    // 失败执行 catch
})
```

### 配置 baseUrl

```js
// 配置请求的基础路径
axios.defaults.baseURL = 'https://api.example.com'
```
